Javascript Terms

Method Chaining aka Cascading

Method chaining in Javascript is the proccess of calling one method after another on an object, in one continuous line of code. When you use method chaining while coding it allows you, the developer perform operations on the object. It reduces the amount of code repetition ,does not interrupt code flow and improves your code readability. If you are more of a visual learner like myself theres an example below.

Arrow Functions

Arrow functions were introduce in ES6. This new function allows to shorten the syntax . The way I would describe its functionality would be instead of being "Javascript create this function and return me the value inside" to "Javascript this is the value now return it". For example without arrow functions to have the console log return "Hello world!" you would need to create a function called hello and have you return the value of "Hello World". With the arrow function you are able to create the function and use the arrow to have the function return its value. To better understand you can view the example below.

Without Arrow Function Code

hello = function() { return "Hello World!"; }

With Arrow Function Code

hello = () => { return "Hello World!"; }
